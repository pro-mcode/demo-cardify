## Overview

Cardify is a payment card management application that allows users to securely add, view, update, and delete payment methods. It also features automatic card provider detection and a feedback form to improve user experience.

## Features

- Add payment method
- View payment methods
- Update payment method
- Update nickname for cards
- Auto-detect card provider based on card number
- Delete payment cards
- Feedback form
- Built with modular React components

## Tech Stack

Frontend:

- React
- JavaScript (ES6+)
- TailwindCSS

Other Tools:

- Git
- React Router
- Vite

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/pro-mcode/demo-cardify.git
cd demo-travely
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```
